Pulseboard

Built 2023 — 2024

Pulseboard is a lightweight status and incident log for small teams. It keeps customers informed with public pages while giving operators webhooks, on-call schedules, and SLAs in one place.

Notable technologies

  • Go backend with clean interfaces for status updates and notifications.
  • SvelteKit front-end for fast dashboards and admin workflows.
  • SQLite for simple, portable persistence that fits small teams.
